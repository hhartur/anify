REST API

Anify API Reference

A fast, cached, multi-source REST API for anime metadata and episode players. No authentication required.

Base URLhttp://localhost:3000
Version2.0.0
FormatJSON
StatusLive
Getting Started
Base URLhttp://localhost:3000

10-min Cache

All responses cached in-memory. Every response includes a cached boolean field.

Multi-source

Switch providers with ?source=id. New sources need zero route changes.

Filters

Filter listings by genres and mode (AND/OR). All combinable.

Authentication

Authentication

This API requires no authentication. All endpoints are publicly accessible with plain GET requests.

No credentials needed

No headers, tokens, or API keys required.

Errors

Error Handling

All errors return a consistent JSON envelope with a single error field.

JSON
{ "error": "Not found" }
Status Codes
200OK — request processed successfully
400Bad Request — missing or invalid parameters
404Not Found — resource does not exist upstream
500Server Error — upstream fetch or parse failure
Multi-source

Sources

Every endpoint accepts ?source=id to select which provider to use. Omit it to use the default source.

GET /sources List all available sources
Response Example
JSON
{
  "sources": [
    { "id": "animesonlinecc", "name": "Animes Online CC" }
  ]
}

 GET /animes

  
List Animes

  
Paginated list of animes. Filter by genre, or switch source.


  

    

      GET
      /animes
      Paginated listing with filters
    

    

      

        
Query Parameters

        
NameTypeRequiredDescription

          
pageintegeroptionalPage number. Default: 1

          
genresstringoptionalComma-separated genre slugs. E.g. acao,seinen. Max 5.

          
modestringoptionalAND (default) — all genres. OR — any genre.

          
sourcestringoptionalSource id. Default: animesonlinecc

        

      

      

        
Response

        

          
FieldTypeDescription

            
sourcestringActive source id

            
cachedbooleanServed from cache

            
pageintegerCurrent page

            
totalPagesintegerTotal pages

            
countintegerItems on this page

            
animes[].slugstringUnique identifier

            
animes[].titlestring?Title (when available)

            
animes[].coverstring?Cover image URL

          

          
JSON


{
  "source": "animesonlinecc",
  "cached": false,
  "page": 1,
  "totalPages": 24,
  "count": 8,
  "animes": [{
    "slug": "vinland-saga",
    "title": "Vinland Saga",
    "cover": "http://localhost:3000/img/...",
  }]
}

        

      

      

        
Try it

        

          
          
          
          
        

        

          

          
Response

        

      

    

  







  
 GET /animes/:slug

  
Get Anime

  
Full anime details including genres, seasons and all episodes.


  

    

      GET
      /animes/:slug
      Details + seasons + episodes
    

    

      

        
Parameters

        
NameTypeRequiredDescription

          
slugstringpathAnime slug (e.g. darwin-jihen)

          
sourcestringoptionalSource id

        

      

      

        
Response

        

          
FieldTypeDescription

            
slugstringIdentifier

            
titlestringTitle

            
imagestringCover image URL

            
descriptionstringSynopsis

            
yearinteger|nullRelease year

            
genres[]array{slug, name} per genre

            
seasonCountintegerNumber of seasons

            
episodeCountintegerTotal episodes

            
seasons[].namestringSeason label

            
seasons[].episodes[]array{slug, number, url}

          

          
JSON


{
  "slug": "darwin-jihen",
  "title": "Darwin Jihen",
  "year": 2023,
  "genres": [
    { "slug": "acao", "name": "Ação" }
  ],
  "seasonCount": 1,
  "episodeCount": 11,
  "seasons": [...]
}

 GET /busca

  
Search

  
Search animes by name, optionally narrowed by genre filter.


  

    

      GET
      /busca
      Textual search with optional genre filter
    

    

      

        
Query Parameters

        
NameTypeRequiredDescription

          
qstringrequiredSearch term, min. 2 characters

          
genresstringoptionalComma-separated genre slugs to filter results

          
modestringoptionalAND or OR. Default: AND

          
sourcestringoptionalSource id

        

      

      

        
Response Example

        
JSON


{

    "year": 2002
